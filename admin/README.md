# GTEC Canteen Admin Dashboard

This is the admin dashboard for the GTEC Canteen system, built with Next.js and Tailwind CSS. It allows canteen administrators to manage orders, update menu items, and view real-time statistics.

## Tech Stack
- **Framework:** Next.js 14
- **UI Library:** React 18
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Real-time:** Socket.io Client
- **Charts:** Recharts

## Getting Started

First, install dependencies:

```bash
npm install
```

Make sure your environment variables are set up. You can copy the `.env.local.example` to `.env.local` and fill in the required values.

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Available Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the app for production.
- `npm run start`: Starts the production server.
- `npm run lint`: Runs ESLint to catch formatting and code quality issues.
